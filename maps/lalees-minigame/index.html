Installation

Step 1: download lalees-minigame.zip containing the map files

Step 2: (optional) Verify the file against this hash 2aaa8a1e030e1e161ef0798bb74177cb0decb7d213a5b9b07200ce234a1fa216 (sha256) [how do I do that?]

Step 3: Extract the contents of the zip file to the Arx Fatalis/Arx Libertatis game directory

Step 4: Start a new game
