Installation

Step 1: download sequencer.zip containing the map files

Step 2: (optional) Verify the file against this hash 7f8c028b36a810e157e575519b67fdcac951aeae6cfdd6b1ae1047912db24642 (sha256) [how do I do that?]

Step 3: Extract the contents of the zip file to the Arx Fatalis/Arx Libertatis game directory

Step 4: Start a new game

The level

Double click on any of the cubes to turn them on or off. When a cube is on it will trigger a sound effect. Pull the lever on the left to enable the sounds or pull it back up to turn it off.
