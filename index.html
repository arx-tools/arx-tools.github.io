Arx Fatalis Mods and Tools

Mods and maps

  • Ambience Gallery — a small debug map to explore all the game's ambiences

Tools

  • arx-level-generator — generates maps for Arx Fatalis
  • arx-convert — converts various Arx Fatalis formats to JSON/YAML and back
  • node-pkware — nodejs implementation of StormLib's pkware compression/decompression algorithm
  • arx-header-size — reads the header size of custom Arx Fatalis file formats that might be partially compressed

Modding tutorials and docs

  • asl-cookbook — practical examples on how to script things in Arx Fatalis using it's script language
