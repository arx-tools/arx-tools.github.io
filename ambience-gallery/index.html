Installation

Step 1: download ambience-gallery.zip containing the map files

Step 2: (optional) Verify the file against this hash 22435e901ac99e5c5b7f5419ebdb700d4d86cf3255dd639de09418c14dd2f44f (sha256) [how do I do that?]

Step 3: Extract the contents of the zip file to the Arx Fatalis/Arx Libertatis game directory

Step 4: Start a new game

The level

Step on any of the blocks to listen to one of the game's ambiences. Step off of the blocks to turn the ambience off.
