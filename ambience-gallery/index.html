Installation

Step 1: download the zip containing the map files

Step 2: Extract the contents of the zip file to the Arx Fatalis/Arx Libertatis game directory

Step 3: Start a new game

The level

Step on any of the tiles to listen to one of the game's ambiences. Step off of the tiles to turn the ambience off.
